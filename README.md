# PassX-Password-manager
A lightweight password manager web app built exclusively with HTML, CSS, and JavaScript, perfect for securely storing personal passwords in browser localStorage. Displays encrypted passwords in a responsive table with one-click copy functionality and a simple, intuitive interface.​

#Key Features
Local Storage: Stores all personal passwords on-device using browser localStorage for complete privacy.​

1.Encrypted Table: Renders passwords in a clean table with masked encryption; reveals on demand.​

2.Copy to Clipboard: Single-click copy from table rows for instant access.​

3.Responsive UI: Fully adapts to mobile, tablet, and desktop with touch-friendly design.​

Technical Stack:-
Pure vanilla HTML structures the form and table, CSS handles responsive styling and visual feedback, while JavaScript manages encryption/decryption, localStorage operations, and dynamic table updates. No external libraries or frameworks required, ensuring fast performance and easy deployment. ​
